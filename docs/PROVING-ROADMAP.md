# ikbi Proving Roadmap

**THE TEST:** Can ikbi create assets that make Peh, Ptah, Luna, Luak, Howa, and the rest of the lab better within the same week?

Each task must create something tangible that another part of the lab immediately benefits from.

---

## Level 1 — Single-Task Reliability ✅ PROVEN

**Tasks:**
- Create a skill module
- Update a README
- Generate documentation
- Fix a small bug
- Add a test
- Refactor a small file

**Goal:** Can ikbi reliably complete bounded tasks?

**Status:** Proven. E2e pipeline promoted twice (all 5 roles green). $0.014/build.
